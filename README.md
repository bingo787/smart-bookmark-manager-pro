# Smart Bookmark Manager Pro

Smart Bookmark Manager Pro 是一款功能强大的 Chrome 浏览器扩展程序，旨在帮助用户高效管理海量书签。它不仅提供了基础的书签查看功能，还集成了重复检测、死链扫描、自动分类以及访问统计等高级功能。

## 🚀 核心功能

- **重复检测**：一键扫描并清理重复的书签链接，保持书签栏整洁。
- **死链检测**：自动检测失效的书签链接（404 或无法访问），防止书签库过期。
- **自动分类**：基于关键词智能识别书签内容，并建议或自动归类到相应文件夹。
- **访问统计**：后台统计各网站的访问频率，直观展示您最常访问的资源。

## 🛠️ 安装指南

1. 下载本项目代码或克隆仓库：
   ```bash
   git clone https://github.com/bingo787/smart-bookmark-manager-pro.git
   ```
2. 打开 Chrome 浏览器，进入插件管理页面：`chrome://extensions/`。
3. 开启页面右上角的 **“开发者模式”**。
4. 点击 **“加载已解压的扩展程序”**，选择本项目所在的文件夹。
5. 在浏览器工具栏中固定插件图标，即可开始使用。

## 📂 项目结构

- `manifest.json`: 插件配置文件，定义权限和入口。
- `popup.html`: 插件弹出界面的 HTML 结构。
- `popup.js`: 处理用户交互、书签操作及核心算法逻辑。
- `background.js`: 后台脚本，负责访问量统计等持续性任务。
- `icons/`: 存放插件图标资源。
